Weather app made in Vue.js 

## Technologies
* Vue.js
* HTML & CSS

## Installation
* Fork or clone this repo ```git clone https://github.com/Nabeel-Siddiqui/simpleWeatherVueApp```
* ```cd``` to ```simpleWeatherVueApp```
* run ```yarn serve```
